eBay Product Search

Aug 21, 2020 2 min read Project

I individually developed a web app that allows users to search items for sale on eBay.com using their API. By typing in keywords and setting filters, users can view the items list that meet their requirements, as well as the detailed information of each product.

The first implementation is a non-responsive design. Technically, I

  • Write Python scripts using Flask which
    • listens to AJAX calls from the front end web page,
    • calls ebay developers APIs to get raw JSON response using a URL REST request,
    • responds frontend webpage with processed JSON text.
  • Design frontend webpage using HTML, CSS, JavaScript, HTML DOM and XMLHTTPRequest object.
  • Deploy the app on Google Cloud Platform App Engine.

Non-responsive design link

The second implementation is a responsive design. Technically, I

  • Write NodeJS scripts which
    • listens to AJAX calls from the front end web page,
    • calls ebay developers APIs to get raw JSON response using a URL REST request
    • responds frontend webpage with processed JSON text.
  • Design frontend webpage using Angular8, implemented two way data binding, page routing and AJAX calls, set styling with Bootstrap to make it compatible with mobile devices.
  • Deploy the app on Google Cloud Platform App Engine.

Responsive design link

Finally, I implement all functions of the web app to a mobile app, which allows users to search through the eBay items catalog and look at the detailed information about them. The App contains 5 screens: Initial Form, Item Catalog, Product Information screen, Seller Information screen and Shipping Information screen. The app has been designed and implemented in a Pixel 2XL emulator by using SDK API 29. Technically, I

  • Develop the android app on AndroidStudio using Java.
  • Design user friendly layouts featured Splash Screen, Navigation Bars, CardView, ViewPager and so on.
  • Implement asynchronous functions including image loading and AJAX calls.
  • Call eBay api on the frontend to provide related images on Internet for each product.
  • Use Node.js on server side, which is deployed on Google Cloud Platform App Engine.
